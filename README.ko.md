# retro-note

잠깐 떠오른 생각을 빠르게 적기 위한, 아늑한 레트로 CRT 터미널. 항상 떠 있어 언제든 손이 닿습니다.

## 소개

retro-note는 옛 CRT 컴퓨터 모습을 그대로 옮긴 한 화면짜리 메모 앱입니다. 켜고, 적고, 저장합니다. 폴더도, 마크다운도, 동기화도 없습니다 — 생각이 날아가기 전에 잠깐 부어두는 공간일 뿐입니다.

메모 앱처럼 보이지만, 사실 익명 채팅도 됩니다 — 코드로 방에 입장해 닉네임 하나로 실시간 대화하세요.

Tauri 2 기반으로 macOS와 Windows에서 네이티브로 동작합니다.

## 기능

- 검정 배경 위 CRT 형광 초록 화면과 기계식 키 입력음
- 항상 위에 떠 있는, 투명하고 테두리 없는 레트로 본체
- 메모 1건당 파일 1개로 Documents 폴더에 `.txt`로 저장
- **채팅** — 코드로 방에 입장해 실시간 대화, 대화 기록 저장 (선택 기능, 계정 필요)

## 단축키

| 동작 | macOS | Windows |
|---|---|---|
| 창 키우기 | `⌘ =` | `Ctrl =` |
| 창 줄이기 | `⌘ -` | `Ctrl -` |
| 기본 크기로 리셋 | `⌘ 0` | `Ctrl 0` |
| 핀치 줌 | `⌘` + 스크롤 | `Ctrl` + 스크롤 |

## 메모는 어디에 저장되나요?

다음 위치에 일반 `.txt` 파일로 저장됩니다:

- **macOS**: `~/Documents/retro-notes/`
- **Windows**: `C:\Users\<사용자명>\Documents\retro-notes\`

파일명 형식: `note_YYYY-MM-DD_HH-MM.txt`

## 채팅

앱을 켜면 화면에 **[ NOTE ]** 와 **[ CHAT ]** 가 뜹니다. CHAT을 고르면:

1. **이메일/비밀번호로 로그인**(또는 회원가입)합니다. 채팅 데이터가 계정에 묶여 방·닉네임·기록이 여러 기기에서 따라옵니다.
2. 닉네임을 설정합니다 (처음 한 번만 물어봅니다).
3. **방 생성**으로 6자리 코드를 발급받거나, 공유받은 코드를 **입력해 입장**합니다.
4. 실시간으로 대화합니다. **메시지가 저장되므로** 다시 들어올 때마다(로그인한 어느 기기에서든) 내가 처음 입장한 시점부터의 기록을 봅니다. 내가 입장하기 전에 오간 메시지는 보이지 않습니다.

방 코드를 아는 사람은 누구나 입장해 자신의 입장 시점 이후 기록을 볼 수 있으므로, 방 코드가 유일한 접근 수단입니다 — 민감한 정보는 주고받지 마세요.
